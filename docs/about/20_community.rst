*********************
Community & Support
*********************

The TBD platform is built by a community of audio researchers, musicians, and developers.
Here's how to get involved, get help, and stay updated.


Get Help
========

- **GitHub Issues** --- Report bugs, request features, or ask questions:
  `dadamachines/ctag-tbd <https://github.com/dadamachines/ctag-tbd/issues>`_

- **Upstream Repository** --- For core DSP engine issues:
  `ctag-fh-kiel/ctag-tbd <https://github.com/ctag-fh-kiel/ctag-tbd/issues>`_


Contribute
==========

Contributions are welcome --- whether it's a new plugin, a bug fix, documentation
improvements, or UI enhancements.

1. Fork the repository on GitHub.
2. Create a feature branch.
3. Submit a pull request with a clear description of your changes.

See :doc:`Plugin Dev --- Getting Started </plugins/getting-started>` to set up your development environment.


Stay Updated
============

- `dadamachines <https://dadamachines.com>`_ --- Official product page and announcements
- `Creative Technologies AG <https://www.creative-technologies.de/>`_ --- Academic research behind CTAG TBD

- `NLnet Project Page <https://nlnet.nl/project/TBD-DSP-Toolkit/>`_ --- Funding and project milestones


Related Projects
================

- `CTAG TBD (upstream) <https://github.com/ctag-fh-kiel/ctag-tbd>`_ --- The original
  open-source Eurorack synthesizer module by Robert Manzke
- `RP2350 Firmware <https://github.com/ctag-fh-kiel/rp2350-arduino-tbd-fw>`_ --- Arduino-based
  firmware for the TBD-16 front-end processor
- `Ableton Link <https://www.ableton.com/en/link/>`_ --- Tempo sync technology integrated
  into TBD-16
