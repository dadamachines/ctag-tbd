Features
========

Beyond its plugin engine, the TBD-16 integrates with the tools musicians already use.

:doc:`Ableton Link <10_ableton_link>`
   Lock tempo and phase with Ableton Live, other Link-enabled apps, or other TBD-16 units over WiFi.

:doc:`USB Audio Interface <20_audio_interface>`
   Stream stereo audio to and from a DAW --- the device shows up as a class-compliant USB audio interface.

:doc:`MIDI <30_midi>`
   Control plugins from external sequencers, keyboards, or DAWs via USB MIDI.
