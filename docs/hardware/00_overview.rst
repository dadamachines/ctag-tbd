********
Hardware
********

dadamachines builds audio DSP hardware for musicians, educators, and product designers.
We offer three paths depending on your project:

.. list-table::
   :header-rows: 1
   :widths: 25 40 35

   * - Product
     - What It Is
     - For Whom
   * - :doc:`TBD-16 <10_tbd16>`
     - Ready-to-use desktop instrument
     - Musicians, performers, educators
   * - :doc:`TBD Main PCB <20_main_pcb>`
     - Core board with 30-pin FFC for custom UI
     - Instrument designers, product developers
   * - :doc:`Custom Integration <30_custom_integration>`
     - ESP32-P4 + RP2350 + Codec on your PCB
     - Manufacturers, OEMs, collaborators

All three tiers run the same CTAG TBD audio engine with 50+ plugins, WiFi control,
MIDI, and Ableton Link.
